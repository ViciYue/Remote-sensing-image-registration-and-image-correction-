
# 图像矫正与遥感影像配准

Coworker @[Pandora-xqq](https://github.com/Pandora-xqq) @Zihang Wang @Yingjian Zhao @Yutong Liu

## （1）运行环境
- **版本**：代码在MATLAB 2022b和2024b以及其他部分版本上均可运行
- **工具**：ImageProcessing Toolbox、Computer Vision Toolbox

## （2）问题简介
- **问题一**：图片矫正
- 实现：仿射变换、投影变换、二阶多项式、三阶多项式、局部加权平均、SIFT、SURF
- **问题二**：遥感影像配准
- 实现：RIFT2、手动控制点投影变换、SIFT、仿射变换、二次多项式变换

## （3）运行
  运行根目录下的 `main.m` 文件即可：
- 单独选择其中的一个方法运行
- `main.m` 文件还提供综合评估的选项
